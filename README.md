# 📊 Student Performance Analysis

## 📌 Project Overview

This project analyzes student performance using Python (Pandas, NumPy, Matplotlib).

## 📂 Dataset

* Student dataset with features like grades, study time, absences, etc.

## 🛠️ Tools Used

* Python
* Pandas
* NumPy
* Matplotlib

## 📊 Key Analysis

* Gender vs performance
* Study time impact
* Absences vs grades
* Correlation between G1, G2, G3

## 📈 Visualizations

* Histograms
* Bar charts
* Scatter plots

## 🧠 Key Insights

* Students with higher study time perform better
* Absences negatively affect grades
* G1 and G2 strongly predict G3

## 🚀 Conclusion

This project demonstrates basic EDA and data visualization skills.
